tbc is a small script for bc containing a few constants and functions.

Currently included are:

CONSTANTS

-Euler's Number, declared as e.
-Pi, declared as pi.
-The golden ratio, declared as phi.

FUNCTIONS

-A floor function, declared as flr(x), and taken from Wikipedia.
-An exponent function, declared as exp(base, exponent).
-A logarithm function, declared as log(exponent, base).
-A nth root function, declared as nrt(x, n).
-A factorial function, declared as fact(x).
